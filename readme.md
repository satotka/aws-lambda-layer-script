# Make AWS Lambda layer

Subject.

1. pip and zip for Layer
2. publish layer to AWS

## Usage

1. ```aws configure``` need to lambda layer role.
2. Edit ```Makefile```, layer-name, library, lib-version, langage

- InfluxDB client :
